## 🔧 КОМАНДЫ ДЛЯ ОБНОВЛЕНИЯ:

```bash
cd /var/www/poizonic-mini-app

# 1. Обновить код
git pull origin main

# 2. Пересобрать frontend
cd frontend
npm run build

# 3. Перезапустить backend
cd ..
pm2 restart poizonic-backend

# 4. Проверить логи в реальном времени
pm2 logs poizonic-backend --lines 0
```

## 🔍 ПРОВЕРКА ПОСЛЕ ОБНОВЛЕНИЯ:

### 1. Откройте сайт в браузере (не в Telegram):

Откройте https://poizonic.ru/ в Chrome/Firefox

### 2. Откройте консоль (F12 → Console)

Вы должны увидеть логи:
- `[index.tsx] Инициализация приложения...`
- `[index.tsx] Приложение запущено`
- `[App] useEffect запущен`
- Через 2 секунды: `[App] Загрузка завершена по таймауту (2 сек)`

### 3. Найдите ошибки (красные сообщения):

Если есть ошибки - скопируйте их и отправьте!

### 4. Проверьте Network tab:

- Все запросы должны быть 200 OK
- Особенно проверьте запрос к `/api/user/init`

---

## ⚠️ ВАЖНО:

**Самое главное - проверьте консоль браузера!** Там будут все ошибки JavaScript, которые блокируют работу приложения.

Если ошибок нет, но загрузка все равно бесконечна - отправьте:
1. Скриншот консоли
2. Скриншот Network tab
3. Что показывают логи PM2 при открытии Mini App
