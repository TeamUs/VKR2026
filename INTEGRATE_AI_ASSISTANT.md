# ИИ-помощник в мини-приложении ВКР

Раздел **FAQ** заменён на **«ИИ-помощник»** (тот же пункт в главном меню). Контент рендерится компонентом `frontend/src/components/AiAssistant.tsx`.

## Быстрая интеграция (Timeweb или любой iframe)

1. Создайте агента / чат с базой знаний у провайдера и получите **URL для встраивания** (страница, которую можно открыть в `<iframe>`).
2. На сервере, **перед** сборкой фронта, задайте переменные окружения:

```bash
# .env.production (рядом с frontend/) или в CI
VITE_AI_ASSISTANT_EMBED_URL="https://ваш-домен/путь-к-виджету"
# опционально:
VITE_AI_ASSISTANT_EMBED_HEIGHT="min(72vh, 640px)"
```

3. Пересоберите фронт:

```bash
cd frontend && npm run build
```

4. Перезагрузите статику (nginx / копирование `dist` — как у вас принято).

Переменные с префикса `VITE_` попадают в бандл **на этапе сборки**. После смены URL нужна **новая** сборка.

## Если чат не отображается

- Убедитесь, что страница **разрешает** открытие в iframe (заголовки `X-Frame-Options` / `Content-Security-Policy` на стороне виджета).
- При необходимости ослабьте атрибут `sandbox` у `<iframe>` в `AiAssistant.tsx` (только осознанно, под ваш виджет).
- Проверьте консоль браузера на ошибки mixed content (HTTP внутри HTTPS).

## Пока URL не задан

В приложении показывается заглушка с инструкцией — это нормально для этапа разработки.

## Связь с менеджером

Кнопка «Написать в Telegram» ведёт на `@poizonic_manager` — при необходимости замените ссылку в `AiAssistant.tsx` (`handleContactManager`).
