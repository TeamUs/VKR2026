# ⚡ БЫСТРЫЙ СТАРТ - Деплой на Production

## 📝 Краткая версия основных шагов

### 1. Подготовка сервера (5-10 минут)

```bash
# Подключение к серверу
ssh root@ваш_сервер_ip

# Обновление системы
sudo apt update && sudo apt upgrade -y

# Установка Node.js 18+
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt-get install -y nodejs

# Установка MySQL
sudo apt install -y mysql-server
sudo systemctl start mysql
sudo systemctl enable mysql

# Установка PM2
sudo npm install -g pm2
pm2 startup  # Выполните выведенную команду

# Установка Nginx
sudo apt install -y nginx
sudo systemctl start nginx
sudo systemctl enable nginx

# Установка Certbot
sudo apt install -y certbot python3-certbot-nginx
```

### 2. Настройка DNS на Reg.ru (2 минуты)

1. Войдите в панель Reg.ru
2. DNS → A-запись:
   - Поддомен: `@` (или `app`)
   - IP: IP вашего сервера
3. Сохраните и подождите 5-30 минут

### 3. Клонирование проекта (2 минуты)

```bash
cd /var/www
sudo mkdir -p poizonic-mini-app
cd poizonic-mini-app
git clone https://github.com/ваш_username/poizonic-mini-app.git
cd poizonic-mini-app_reserve_copy
```

### 4. Настройка базы данных (2 минуты)

> **Если у вас уже есть БД:** просто проверьте подключение и пропустите создание!

```bash
# Проверка подключения к существующей БД
mysql -u gen_user -p -h localhost default_db
# В MySQL: SHOW TABLES; EXIT;

# Если БД нужно создать (только если её нет):
sudo mysql -u root -p
# В MySQL консоли:
# CREATE DATABASE default_db CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
# CREATE USER 'gen_user'@'localhost' IDENTIFIED BY 'ваш_пароль';
# GRANT ALL PRIVILEGES ON default_db.* TO 'gen_user'@'localhost';
# FLUSH PRIVILEGES;
# EXIT;
```

### 5. Настройка Backend (3 минуты)

```bash
cd backend
npm install

# Создайте .env файл
cp env.example .env
nano .env  # Заполните все переменные

# Создайте директорию для загрузок
mkdir -p uploads/reviews
```

### 6. Сборка Frontend (2 минуты)

```bash
cd ../frontend
npm install
npm run build
```

### 7. Запуск через PM2 (2 минуты)

```bash
cd ..
# Скопируйте ecosystem.config.js.example в ecosystem.config.js
cp ecosystem.config.js.example ecosystem.config.js
nano ecosystem.config.js  # Измените путь к проекту, если нужно

mkdir -p logs
pm2 start ecosystem.config.js
pm2 save
```

### 8. Настройка Nginx (5 минут)

```bash
# Скопируйте конфигурацию
sudo cp nginx.conf.example /etc/nginx/sites-available/poizonic

# Отредактируйте, заменив вашдомен.ru на ваш домен
sudo nano /etc/nginx/sites-available/poizonic

# Активируйте конфигурацию
sudo ln -s /etc/nginx/sites-available/poizonic /etc/nginx/sites-enabled/
sudo rm /etc/nginx/sites-enabled/default
sudo nginx -t
sudo systemctl reload nginx
```

### 9. Установка SSL (2 минуты)

```bash
sudo certbot --nginx -d вашдомен.ru -d www.вашдомен.ru
```

### 10. Настройка Telegram Bot (2 минуты)

1. Откройте @BotFather в Telegram
2. `/mybots` → выберите бота
3. Bot Settings → Domain → введите: `вашдомен.ru`
4. Сохраните

## 🆘 Частые проблемы

### Backend не запускается
```bash
pm2 logs poizonic-backend --err
cat backend/.env  # Проверьте переменные окружения
```

### 502 Bad Gateway
```bash
pm2 status  # Проверьте, запущен ли backend
curl http://localhost:3000  # Проверьте доступность
```

### Frontend не загружается
```bash
ls -la frontend/dist/  # Проверьте наличие файлов
sudo nginx -t  # Проверьте конфигурацию Nginx
```
