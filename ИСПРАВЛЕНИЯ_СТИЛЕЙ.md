## Что нужно сделать на сервере:

### Шаг 1: Запустить диагностический скрипт
```bash
cd /var/www/poizonic-mini-app
chmod +x diagnose-server.sh
./diagnose-server.sh > diagnose-output.txt 2>&1
cat diagnose-output.txt
```

### Шаг 2: Обновить код
```bash
cd /var/www/poizonic-mini-app
git pull origin main
```

### Шаг 3: Переустановить зависимости frontend
```bash
cd /var/www/poizonic-mini-app/frontend
rm -rf node_modules package-lock.json
npm ci
```

### Шаг 4: Проверить версию styled-components
```bash
cd /var/www/poizonic-mini-app/frontend
npm list styled-components
# Должно показать: styled-components@6.1.13
```

### Шаг 5: Очистить и пересобрать frontend
```bash
cd /var/www/poizonic-mini-app/frontend
rm -rf dist .vite
npm run build
```

### Шаг 6: Проверить собранные файлы
```bash
cd /var/www/poizonic-mini-app/frontend/dist
ls -la
# Проверить наличие index.html и папки assets

# Проверить размер assets
du -sh assets/

# Проверить наличие стилей в JS файлах
find assets -name "*.js" -exec grep -l "styled-components\|GlobalStyle" {} \; | head -3
```

### Шаг 7: Проверить Node.js версию на сервере
```bash
node -v
# Должна быть версия 18.x или выше

npm -v
```

### Шаг 8: Перезагрузить Nginx
```bash
sudo systemctl reload nginx
```

### Шаг 9: Проверить сайт в браузере
1. Откройте сайт на сервере
2. Откройте консоль браузера (F12)
3. Проверьте:
   - Есть ли ошибки в консоли
   - Есть ли тег `<style data-styled="active">` в `<head>`
   - Есть ли содержимое в этом теге (не пустой)
   - Правильно ли применяются стили

## Дополнительные проверки:

### Проверить логи Nginx
```bash
sudo tail -f /var/log/nginx/error.log
```

### Проверить процесс сборки (если есть ошибки)
```bash
cd /var/www/poizonic-mini-app/frontend
npm run build 2>&1 | tee build.log
```

### Если стили все еще не работают:

1. **Проверить браузер консоль на ошибки:**
   ```javascript
   // В консоли браузера выполните:
   console.log('Style tags:', document.querySelectorAll('style').length);
   const styleTag = document.querySelector('style[data-styled]');
   console.log('Styled tag:', styleTag);
   console.log('Style content length:', styleTag?.textContent?.length || 0);
   console.log('GlobalStyle injected:', styleTag?.textContent?.includes('GlobalStyle') || false);
   ```

2. **Проверить версию Node.js:**
   - Локально: `node -v`
   - На сервере: `node -v`
   - Должны совпадать (или быть близкими версиями)

3. **Проверить права на файлы:**
   ```bash
   ls -la /var/www/poizonic-mini-app/frontend/dist/
   # Файлы должны быть читаемы для nginx пользователя
   ```

4. **Очистить кеш браузера:**
   - Hard refresh: Ctrl+Shift+R (Windows/Linux) или Cmd+Shift+R (Mac)

## Если проблема сохраняется:

Запустите полную диагностику:
```bash
cd /var/www/poizonic-mini-app
./diagnose-server.sh > full-diagnosis.txt 2>&1
# Отправьте содержимое full-diagnosis.txt для анализа
```
