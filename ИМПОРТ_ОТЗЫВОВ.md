# 📥 ИНСТРУКЦИЯ ПО ИМПОРТУ ОТЗЫВОВ

## 📁 Где находится файл для отзывов?

**Файл:** `backend/reviews-data.json`

Это JSON-файл, в который нужно заносить отзывы перед импортом в базу данных.

---

## 📋 Структура файла reviews-data.json

```json
[
  {
    "username": "@username",
    "fullName": "Имя Фамилия",
    "text": "Текст отзыва",
    "date": "2025-01-15 12:00:00",
    "photoUrls": [
      "/uploads/reviews/photo1.jpg",
      "/uploads/reviews/photo2.jpg"
    ]
  }
]
```

### Поля:
- **`username`** (обязательно) — Telegram username (например, `@username`)
- **`fullName`** (опционально) — полное имя пользователя
- **`text`** (обязательно) — текст отзыва
- **`date`** (опционально) — дата отзыва в формате `YYYY-MM-DD HH:mm:ss`. Если не указано, используется текущая дата
- **`photoUrls`** (опционально) — массив путей к фотографиям. Может быть пустым `[]`

---

## 🚀 Как импортировать отзывы?

### 1. Заполните `backend/reviews-data.json`
   - Добавьте все отзывы в массив
   - Убедитесь, что JSON валидный

### 2. Загрузите фотографии (если есть)
   - Поместите фотографии в папку `backend/uploads/reviews/`
   - Используйте эти пути в поле `photoUrls`

### 3. Запустите скрипт импорта:
```bash
cd backend
node import-reviews.js
```

---

## ⚙️ Что делает скрипт?

1. ✅ Читает `reviews-data.json`
2. ✅ Подключается к базе данных
3. ✅ Для каждого отзыва:
   - Создает пользователя (если его нет) или использует существующего
   - Создает отзыв с рейтингом **5 звезд**
   - Автоматически одобряет отзыв (`is_approved = 1`)
   - Добавляет все фотографии из массива `photoUrls`
4. ✅ Выводит статистику (сколько импортировано, сколько пропущено)

## 📝 Пример заполнения

```json
[
  {
    "username": "@ivanov",
    "fullName": "Иван Иванов",
    "text": "Отличный сервис! Всё пришло быстро и в хорошем состоянии.",
    "date": "2025-01-10 14:30:00",
    "photoUrls": [
      "/uploads/reviews/review-ivanov-1.jpg",
      "/uploads/reviews/review-ivanov-2.jpg"
    ]
  },
  {
    "username": "@petrov",
    "fullName": null,
    "text": "Быстрая доставка, рекомендую всем!",
    "date": "2025-01-12 18:00:00",
    "photoUrls": []
  }
]
```

---

## ⚠️ Важно

- **Рейтинг всегда 5 звезд** (как вы указали)
- **Отзывы автоматически одобряются** (появляются сразу в приложении)
- **Пользователи создаются автоматически**, если их нет в БД
- **Фотографии должны быть загружены** в `backend/uploads/reviews/` перед импортом

---

## 🔄 Если нужно обновить отзывы

1. Отредактируйте `reviews-data.json`
2. Запустите `node import-reviews.js` снова
3. Скрипт создаст новые записи (дубликаты не проверяются)
