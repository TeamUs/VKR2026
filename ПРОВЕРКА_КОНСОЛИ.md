# 🔍 ПРОВЕРКА КОНСОЛИ БРАУЗЕРА

## ❗ ВАЖНО: Проверьте консоль браузера!

Проблема скорее всего в JavaScript ошибке. Нужно проверить консоль.

### Способ 1: В обычном браузере (проще)

1. Откройте https://poizonic.ru/ в Chrome/Firefox
2. Нажмите **F12** (откроется DevTools)
3. Перейдите на вкладку **Console**
4. Найдите **КРАСНЫЕ ОШИБКИ**
5. Скопируйте и отправьте их

### Способ 2: В Telegram (сложнее)

Если открываете через Telegram Mini App:
- В некоторых версиях Telegram можно открыть DevTools
- Или используйте `window.console.log` для отладки

## 🔍 ЧТО ИСКАТЬ В КОНСОЛИ:

1. **JavaScript ошибки** (красные):
   - `Uncaught TypeError: ...`
   - `Uncaught ReferenceError: ...`
   - `Cannot read property ... of undefined`

2. **Ошибки загрузки ресурсов**:
   - `Failed to load resource`
   - `404 Not Found`
   - `CORS error`

3. **Ошибки API**:
   - `Failed to fetch`
   - `Network request failed`

## 🔧 ВРЕМЕННОЕ РЕШЕНИЕ:

Я добавил обработку ошибок и логирование. Обновите код:

```bash
cd /var/www/poizonic-mini-app
git pull origin main
cd frontend
npm run build
pm2 restart poizonic-backend
```

После обновления:
1. Откройте сайт в браузере
2. Откройте консоль (F12)
3. Посмотрите логи с префиксом `[App]` и `[index.tsx]`
4. Найдите ошибки

---

## 📝 ОЖИДАЕМЫЕ ЛОГИ:

В консоли должны быть:
- `[index.tsx] Инициализация приложения...`
- `[index.tsx] Приложение запущено`
- `[App] useEffect запущен`
- `[App] Загрузка завершена по таймауту (2 сек)` - через 2 секунды

Если этих логов нет - значит есть ошибка, которая блокирует выполнение!
