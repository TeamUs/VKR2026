# Настройка автоматической публикации отзывов в Telegram канал

## Описание

Теперь при создании отзыва через мини-приложение, он автоматически публикуется в ваш Telegram канал `@poizonic_feedback` от имени пользователя.

## Настройка канала

### 1. Добавьте бота как администратора канала

1. Откройте канал `@poizonic_feedback`
2. Перейдите в "Управление каналом" → "Администраторы"
3. Добавьте бота `@poizonic_bot` как администратора
4. Дайте боту права:
   - ✅ Отправлять сообщения
   - ✅ Редактировать сообщения других пользователей (опционально)

### 2. ID канала настроен

✅ **ID канала:** `-1002499442701`  
✅ **Канал:** `@poizonic_feedback`  
✅ **Настройка завершена**

## Формат публикуемых отзывов

### Без фото:
```
⭐ Оценка: ⭐⭐⭐⭐⭐ (5/5)

📝 Отзыв:
Отличный сервис! Быстро и качественно.

👤 От: Иван Петров
```

### С фото:
- Фото публикуется с подписью в том же формате
- Подпись содержит всю информацию об отзыве

## Технические детали

### Функции в `server.js`:

1. **`sendReviewToFeedbackChannel(reviewData, userInfo)`** - основная функция отправки
2. **`sendTelegramPhoto(chatId, photoUrl, caption)`** - отправка фото с подписью
3. **`sendTelegramMessage(chatId, message)`** - отправка текстового сообщения

### Переменные окружения:

1. Создайте тестовый отзыв через мини-приложение
2. Проверьте, что отзыв появился в канале
3. Убедитесь, что формат сообщения корректный
4. Проверьте отправку фото (если есть)

## Возможные проблемы

### Бот не может публиковать в канале
- Проверьте, что бот добавлен как администратор
- Убедитесь, что у бота есть права на публикацию

### Фото не загружаются
- **Проблема:** `localhost` URL недоступен для Telegram
- **Решение:** Настройте домен с SSL сертификатом
- **Временное решение:** Отзывы публикуются без фото (fallback на текст)
- **После настройки домена:** Обновите `FRONTEND_URL` в переменных окружения

### Ошибки API Telegram
- Проверьте токен бота
- Убедитесь, что канал существует и бот имеет доступ
