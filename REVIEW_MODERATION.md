# Система модерации отзывов

## Описание

Добавлена система модерации отзывов для защиты от спама и некачественного контента. Теперь все отзывы проходят проверку перед публикацией в канале.

## Как это работает

### 1. Создание отзыва
- Пользователь создает отзыв через мини-приложение
- Отзыв сохраняется в базу данных со статусом `is_approved = 0` (не одобрен)
- Менеджер получает уведомление о новом отзыве с ID для модерации
- Отзыв НЕ публикуется в канале автоматически

### 2. Модерация
- Администратор просматривает отзывы в админ-панели
- Может одобрить или удалить отзыв
- При одобрении отзыв автоматически публикуется в канале
- При удалении отзыв полностью удаляется из базы данных

## Структура базы данных

### Новые поля в таблице `reviews`:
```sql
- is_approved TINYINT(1) DEFAULT 0     -- 0 = не одобрен, 1 = одобрен
- moderated_at TIMESTAMP NULL         -- время одобрения
- moderated_by VARCHAR(50) NULL       -- ID модератора
```

## API эндпоинты для модерации

### 1. Получение отзывов для модерации
```
GET /api/admin/reviews?status=pending&page=1&limit=10
```

**Параметры:**
- `status`: `all`, `pending`, `approved`
- `page`: номер страницы
- `limit`: количество на странице

**Ответ:**
```json
{
  "reviews": [...],
  "total": 25,
  "page": 1,
  "limit": 10,
  "totalPages": 3,
  "status": "pending"
}
```

### 2. Одобрение отзыва
```
POST /api/admin/reviews/{reviewId}/approve
```

**Тело запроса:**
```json
{
  "moderatorId": "admin"
}
```

**Результат:**
- Отзыв помечается как одобренный
- Автоматически публикуется в канале
- Менеджер получает уведомление

### 3. Удаление отзыва
```
DELETE /api/admin/reviews/{reviewId}
```

**Результат:**
- Отзыв полностью удаляется из базы данных
- Менеджер получает уведомление

### 4. Статистика отзывов
```
GET /api/admin/reviews/stats
```

**Ответ:**
```json
{
  "total": {
    "total_reviews": 50,
    "approved_reviews": 45,
    "pending_reviews": 5,
    "average_rating": 4.7
  },
  "daily": [
    {
      "date": "2024-01-15",
      "total": 3,
      "approved": 2
    }
  ]
}
```

## Обновленные эндпоинты

### Публичные эндпоинты (для пользователей):
- `GET /api/reviews` - показывает только одобренные отзывы
- `GET /api/reviews/average-rating` - считает только одобренные отзывы

### Админские эндпоинты:
- `GET /api/admin/reviews` - все отзывы для модерации
- `POST /api/admin/reviews/:id/approve` - одобрить отзыв
- `DELETE /api/admin/reviews/:id` - удалить отзыв
- `GET /api/admin/reviews/stats` - статистика

## Безопасность

### Защита от спама:
- ✅ Все отзывы проходят модерацию
- ✅ Неодобренные отзывы не видны пользователям
- ✅ Возможность удаления спама

### Уведомления:
- ✅ Менеджер получает уведомления о новых отзывах
- ✅ Уведомления об одобрении/удалении
- ✅ ID отзыва для быстрой модерации

## Админ-панель

Для полноценной работы нужно добавить в админ-панель раздел "Отзывы" с:
- Списком отзывов с фильтрацией (все/ожидающие/одобренные)
- Кнопками "Одобрить" и "Удалить"
- Статистикой отзывов
- Предпросмотром фото (если есть)

## Логирование

Все действия модерации логируются:
- Создание отзыва
- Одобрение отзыва
- Удаление отзыва
- Публикация в канале
