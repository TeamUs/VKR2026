# 🚀 КАК ЗАПУСТИТЬ ПРОВЕРКУ PRODUCTION

## 📍 На сервере:

```bash
cd /var/www/poizonic-mini-app
node scripts/check-production-env.js
```

## 📍 Локально (на вашем компьютере):

```bash
cd poizonic-mini-app_reserve_copy
node scripts/check-production-env.js
```

---

## ✅ Если всё ОК:

Вы увидите:
```
🔍 Проверка переменных окружения для production...

✅ Все проверки пройдены! Можно деплоить в production.
```

---

## ❌ Если есть ошибки:

Вы увидите что-то вроде:
```
🔍 Проверка переменных окружения для production...

❌ ОШИБКИ:
   ❌ FRONTEND_URL содержит localhost: http://localhost:3000
   Используйте реальный домен (например: https://poizonic.ru)

❌ Проверка не пройдена! Исправьте ошибки перед деплоем.
```

---

## 💡 Полезно запускать:

- ✅ Перед деплоем на production
- ✅ После изменения .env файла
- ✅ Перед push в git
